# Saras AI Bot

A private AI chatbot project focused on building a fast, intelligent, and modern conversational assistant for customer support and business interactions.

## Tech Stack

### Frontend
- React
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL

### AI
- Claude 3 Haiku
- Voyage AI

## Features

- AI-powered conversations
- Context-aware responses
- Conversation memory
- Retrieval-Augmented Generation (RAG)
- Semantic search using embeddings
- Responsive chat interface
- REST API backend
- PostgreSQL data storage

## Goals

- Build a production-ready AI chatbot
- Experiment with RAG and prompt engineering
- Improve response quality and latency
- Create a scalable chatbot architecture
- Explore AI-powered customer support workflows
